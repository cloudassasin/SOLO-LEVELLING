An immersive dark-themed Solo Leveling fan site built with **Next.js**, **React**, and **Tailwind CSS**, inspired by the System UI, Gates, and Hunters.

## Tech Stack

- **Framework**: Next.js (App Router, TypeScript-ready)
- **UI**: React + Tailwind CSS
- **Design**: Dark, slate-based palette with electric blue (\`#00d4ff\`) accents

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open \`http://localhost:3000\` in your browser to explore the site.

## Structure

- \`app/layout.tsx\`: Root layout and global shell.
- \`app/page.tsx\`: Main page with Hero, Storyline, Character Cards, and Shadow Army Gallery.
- \`app/globals.css\`: Global styles and System-inspired utility classes.
- \`tailwind.config.ts\`: Tailwind configuration with extended electric blue tokens.
